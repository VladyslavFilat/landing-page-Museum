## 📌 About the project
This project is a responsive landing page designed for a museum.
It focuses on clean layout, smooth interactions, and modern UI structure.

The main goal was to practice real-world frontend development skills including layout building, JavaScript interactivity, and project structuring.

---

## 🚀 Getting Started
  To get a local copy up and running, follow these simple steps.


  ### Clone the repository:

    git clone https://github.com/VladyslavFilat/landing-page-Museum.git

  ### Open the project folder:

    cd landing-page-Museum

  ### Install NPM packages:

    npm install

   ### Start the project:

    npm start

---

## ✨ Features

- Responsive design (mobile / tablet / desktop)
- Custom image slider with autoplay
- Dot navigation for slides
- Modular JavaScript structure
- Clean SCSS architecture
- Parcel bundler setup

---

## 🖼 Slider functionality

The slider is implemented using vanilla JavaScript without external libraries.

It includes:
- automatic slide switching (autoplay)
- manual navigation via dots
- infinite loop effect
- smooth CSS transitions

---

## 💻 Technologies Used

  - HTML5 — semantic structure
  - SCSS (Sass) — styles
  - JavaScript (ES6 modules)
  - Parcel — bundler
  - Git / GitHub Pages — version control & deployment
